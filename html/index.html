Interactive Dynamic Volume Illumination with Refraction and Caustics

Description

This program is an implementation of the paper "Interactive Dynamic Volume Illumination with Refraction and Caustics" by Jens G. Magnus and Stefan Bruckner.

Program

The program was implemented in a C++ framework, uses OpenGL 4.4 and therefore needs a Nvidia platform.

Camera interaction

By pressing "Tab", the interaction mode can be switched between mouse movement and UI interaction. The mouse movement mode is active, if no mouse cursor is visible.

User interface

The user interface was implemented based on AntTweakBar library and has several sections.

General

The general section provides a drop down menu to chose the loaded model (see Section Dataset) and the number of slices used for rendering. The default model is the skull of a two-toed sloth. Additional models can be loaded with the button "Load additional meshes" but this can take a few minutes.

Light

In this section parameters for the light setting can be adapted. The values for the specular reflection constant and the shininess constant can be edited.

Volume TF & Medium TF

The transfer function for volume and medium color are defined by four control points. The corresponding RGB and alpha values can be changed. For the medium color, the alpha value won't have any impact for the rendering calculation. The change will be applied once the button "Apply" is pressed to avoid unnecessary calculations. The "Reset" button will reset the values of the transfer function to the inital settings.

Medium TF

The transfer function for the medium color has also four control points. The RGB colors can be changed

Dataset

The volume data used for this program was taken from Digital Morphology library, an archive providing high-resolution X-ray computed tomography of biological specimens.

The following datasets have been chosen for the program and can be loaded:
  • Alligator mississippiensis - American Alligator
  • Aspidites melanocephalus - Black-headed Python
  • Choloepus hoffmanni - Two-toed Sloth (default model)
  • Elseya dentata - Northern Snapping Turtle
  • Lama glama - Llama
